Token Login

            
For direct token login, you can use the /token-login route with the following query parameters:

            
    
                
  • token: The JWT token
    • 
                
  • userData: URL-encoded JSON string of the user data
    • 
            

            
Example URL:

            
http://localhost:3000/token-login?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...&userData=%7B%22_id%22%3A%226884ff30405643f0bfc65d23%22%2C%22name%22%3A%22Admin%20User%22%2C%22email%22%3A%22user%40example.com%22%2C%22role%22%3A%22admin%22%7D
